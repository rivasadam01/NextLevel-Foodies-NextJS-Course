# NextLevel Food

NextLevel Food is a Next.js-based web application for food enthusiasts to explore, share, and connect over delicious meals.

## Features

- Dynamic image slideshow on the homepage
- Community section for users to join and interact
- Meal exploration page

## Technologies Used

This project leverages several modern web development technologies and concepts:

- **Next.js 14**: For server-side rendering, routing, and API routes
- **React**: For building user interfaces
- **Server Actions**: For handling form submissions and server-side logic
- **Server-Side Components**: For improved performance and SEO
- **Client-Side Components**: For interactive UI elements
- **CSS Modules**: For component-scoped styling
- **Image Optimization**: Using Next.js Image component
- **Dynamic Routes**: For meal detail pages
- **API Routes**: For handling backend logic
- **Form Validation**: Both client-side and server-side
- **Error Handling**: Using Next.js error boundaries
- **Metadata API**: For dynamic SEO optimization
- **Amazon S3**: For storing and serving static assets like images

## Getting Started

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## Project Structure

- `app/page.js`: Main homepage component
- `components/images/image-slideshow`: Component for the image slideshow
- `app/page.module.css`: Styles for the homepage
